WicketPixie. Reloaded.

Welcome to the home page of WicketPixie's fork!

Some of you might wonder: why fork? Well, WicketPixie 1.2 was abandoned, and 2.0 is not finished yet. Until it is, we should have something nice to use, shouldn't we? Enter WiPi 1.x.

I added some nice features, such as internationalization support. However, this fork requires WordPress 3.0 or higher, since some new features (such as custom nav menus) use functions added in this version.

What you see here is the HTML and CSS of this very fork, extracted, stripped and adapted to serve the purposes of this page.

For more info, check out the ReadMe and the upgrading instructions included in the theme.
